基线栅格 Baseline Grids

所有控件的基本元素都对齐到 8dp 宽的栅格，字体则对齐到 4dp 的栅格，工具栏图标也是对齐到 4dp 的栅格。这个设计应用于移动设备、平板和桌面。

触摸元素大小 Touch Target Size

最小的触摸元素大小是 48dp。尤其是在使用 24dp 的图标和 40dp 的头像时，请保持其可触摸大小为 48dp。触摸元素之间也不应该互相重叠。

一般情况下，48dp 在屏幕上大概是 9mm 大，这样用户使用手指触摸操作会比较容易且精确。

如果您设计的触摸元素无法达到 48dp 的大小，那么至少请保持以下标准:

  • 触摸元素的大小至少在任何屏幕上都不能小于 7mm。
  • 请保持您应用 UI 元素的密度和清晰度。
